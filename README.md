# belitrophy-app
BeliTrophy adalah aplikasi web yang menampilkan katalog trophy dan menyediakan pemesanan langsung tanpa akun. Pengguna dapat memilih banyak jenis trophy sekaligus dan menentukan jumlah untuk setiap produk. Ketika pesanan dikirim, data akan masuk ke server melalui API khusus dan disimpan sebagai pesanan baru.  Sementara itu, admin memiliki akses ke dashboard yang berada pada URL khusus dan dilindungi login. Dari halaman admin, produk dapat dikelola melalui fitur CRUD, dan setiap pesanan yang dibuat user dapat dipantau serta diperbarui statusnya. Struktur ini memisahkan peran user dan admin secara aman meskipun berada dalam satu aplikasi.
